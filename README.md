# Vernam CLI Tool

Perfect secrecy cipher algorithm with truly random key implemented in C

# Install
```sh
$ make
```

# Usage
```sh
$ ./vernam
```

# Features
- Support command line arguments to encrypt
- Creation of thousands of key files at one time
